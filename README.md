# After-sales-service
"A responsive 5-page website built with HTML, CSS, JavaScript, and Tailwind CSS. It captures user name and email and stores them securely in a database using PHP and XAMPP. Designed for after-sales vehicle services, the system efficiently manages customer inquiries and support records."
